# GitHub Copilot Usage

## Objective

Accelerate development of backend and frontend code.

## Example Usage

While writing the FastAPI endpoints, Copilot suggested code structures for:

* API route definitions
* request handling
* JSON response formatting

Example endpoint implemented:

POST /query

which receives a natural language question and returns the computed result.

Reduced boilerplate coding and sped up API development.
